# Z 函数

对于一个长度为 $n$ 的字符串 $S$，定义函数 $Z[i]$ 表示 $S$ 与其以 $i$ 位置开始的后缀部分 $S[i,\ n)$ 的**最长公共前缀**的长度。

+   特殊地，$Z[0]$ 的值一般规定为 $0$ 或 $n$。

## Z 算法

线性求解 Z 函数的算法称为 Z 算法。

Z 算法通过遍历 $S$ 的同时维护一个与其**前缀相等的最右子串**，通过该子串的信息和已经求得的 Z 函数快速求得剩余每个位置的 Z 函数。~~算法过程类似于 Manacher。~~

从左到右依次遍历每个位置时，根据位置 $i$ 是否在维护的区间 $[l,\ r]$ 内分情况进行讨论。

令 $len = r - l + 1,\ S_1 = S[0,\ len - 1],\ S_2 = S[l,\ r]$，满足 $S_1 = S_2$。

+   当 $i$ 在 $S_2$ 内部时，令 $j = i - l$ 作为位置 $i$ 在 $S_1$ 中的映射。

    由于 $S_1 = S_2$，则 $i$ 位置的 $Z$ 函数可以根据情况从 $j$ 处**继承**一部分。

    +   若 $j + Z[j] < len - 1$，说明从 $j$ 位置开始匹配的长度不足以到达 $S_1$ 的右端点，则从 $i$ 位置开始匹配的长度也不足以到达 $S_2$ 的右端点。得到 $Z[i] = Z[j]$，无法进一步拓展。

        ```cpp
        if (j + z[j] < len - 1)
            z[i] = z[j];
        ```

    +   若 $j + Z[j] \geq len - 1$，说明从 $i$ 位置开始匹配的长度至少可以到达 $S_2$ 的右端点，得到 $i + Z[i] \geq r$，更右侧的部分则依次遍历求得匹配终点。

        求解完成后，更新最右子串信息。

        ```cpp
        if (j + z[j] >= len - 1)
        {
            z[i] = r - i;
            while (i + z[i] < n && s[z[i]] == s[i + z[i]])
            {
                ++z[i];
            }
            l = i;
            r = i + z[i] - 1;
        }
        ```

+   若 $i$ 在 $S_2$ 的右侧，则没有可利用的最右子串信息，直接遍历求得并更新最右子串信息。

    ```cpp
    if (i > r)
    {
        z[i] = 0;
        while (i + z[i] < n && s[z[i]] == s[i + z[i]])
        {
            ++z[i];
        }
        l = i;
        r = i + z[i] - 1;
    }
    ```

实际上，无论 $i$ 在哪个位置，都可以首先尝试继承长度，然后尝试向右匹配，从而减少代码量。

注意到遍历位置的次数等于最右子串右端点右移的次数，恰好为 $n$，因此 $Z$ 算法**时间复杂度**为 $O(n)$。

### 模板

```cpp
class Z
{
public:
    static std::vector<int> calc(const std::string& s)
    {
        int n{ static_cast<int>(s.length()) };
        std::vector<int> zs(n);

        // TODO：设置Z[0]默认值
        zs[0] = ;
        int l, r{ -1 };
        for (int i{ 1 }; i < n; ++i)
        {
            if (i <= r)
                zs[i] = std::min(zs[i - l], r - i);
            while (i + zs[i] < n && s[zs[i]] == s[i + zs[i]])
            {
                ++zs[i];
            }
            if (i + zs[i] - 1 > r)
            {
                l = i;
                r = i + zs[i] - 1;
            }
        }
        return zs;
    }
};
```
